# rpg-tormenta-app

Monorepo base para app mobile (Expo/React Native) + API (FastAPI).

## Estrutura
- `apps/mobile`: app Expo (frontend)
- `services/api`: API FastAPI (backend)
- `packages/shared`: tipos/validacoes compartilhadas (opcional)
